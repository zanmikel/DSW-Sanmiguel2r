HTML

HTML es un lenguaje de marcado que se utiliza para el desarrollo de páginas de Internet. Se trata de la siglas que corresponden a HyperText Markup Language, es decir, Lenguaje de Marcas de Hipertexto.

Principales Ventajas del HTML

● La estructura del texto es amena.

● Es pequeño el volumen de información.

● No se requiere de grandes conocimientos, mientras se tenga un editor de web.

● Permite todos los exploradores.

Estructura de una Pagina Web

● Desarrollar un proyecto de sitio (determinar cuántas páginas web se incluirán en el sitio, cuál será su tema y cómo se relacionarán).

● Preparar una carpeta (electrónica) en la que se almacenarán todos los materiales.

● La página web del título generalmente se guarda con el nombre index.htm o welcome.htm.

Definicion de CSS

CSS, en español «Hojas de estilo en cascada», es un lenguaje de diseño gráfico para definir y crear la presentación de un documento estructurado escrito en un lenguaje de marcado.​
